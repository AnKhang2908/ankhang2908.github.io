Grammar Rules Guide

1. Comparative Adverbs (2 things)

Use when comparing two actions:

  • Short adverbs: add -er (fast → faster, hard → harder)
  • -ly adverbs: use more (carefully → more carefully)
  • Irregular: well → better, badly → worse

2. Superlative Adverbs (3+ things)

Use when comparing three or more actions:

  • Short adverbs: use the -est (fast → the fastest)
  • -ly adverbs: use the most (carefully → the most carefully)
  • Irregular: well → the best, badly → the worst
